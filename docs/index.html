Project CIRCL Vanhout

This project aims at helping first-time home-builders imagine and configure their dream house by providing 3D visualisations.

It was created in partnership with the start-up CIRCL by Groep Vanhout during Open Summer of Code 2022, an initiative of Open Knowledge Belgium

The project

Renovating/building your home often incorporates a lot of additional time, money and stress. CIRCL, a startup of Vanhout, aims at optimizing this process by calculating the optimal house plans according to the needs of the client.

In its current state, CIRCL already has software for calculating optimal plans and visualizing them in 2D. This project by Team Sphere aims at visualizing the plans in 3D to give future home builders a better insight into what their future home could look like.

Try UX Design

We implemented an application that allows the user to type an address into a form. After that an optimal plan, calculated by CIRCL, is visualized in 3D.

The user can view the building from multiple angles by pressing the left mouse button and dragging the mouse in different directions.

It is also possible to zoom in and out using the scroll wheel of the mouse. Additionally, the user can view the separate floors with the slider that is provided in the user interface.
